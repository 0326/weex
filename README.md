# Weex App Demo

## 环境配置
1. 买一台Mac;
2. 安装weex-toolkit: https://github.com/weexteam/weex-toolkit;
3. 安装weex开发环境: https://github.com/alibaba/weex;

环境依赖安装可能比较慢，建议灵活切换cnpm和翻墙工具。

## 快速开始
```shell
git clone git@github.com:0326/weex.git
git checkout origin/0326/dev
npm i 
./start
```
浏览器访问：http://localhost:12580/ ，在src/文件夹下面开始编写你的代码吧！
